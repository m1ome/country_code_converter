# CodeConverter

Module for converting `iso2`, `iso3` and `ioc` codes back and forth.

## Installation

The package can be installed by adding `country_code_converter` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:country_code_converter, "~> 1.0.0"}
  ]
end
```
